# Screenshots

These screenshots were captured from the real FinCore Ledger application running locally against the UCI Online Retail public dataset demo seed on July 19, 2026 (Asia/Karachi).

## Captures

- `login-development-accounts.png` - public dataset demo identity selector.
- `dashboard-financial-overview.png` - GBP wallet balance, public invoice payment volume, and recent UCI payments.
- `transactions-activity.png` - UCI invoice-derived payments backed by the API.
- `withdrawal-review-queue.png` - merchant withdrawal request queue with dataset proceeds.
- `ledger-journal-explorer.png` - public dataset ledger entries and balanced postings.
- `reconciliation-review.png` - completed reconciliation run with matched wallet projections.
- `webhook-operations.png` - merchant webhook endpoint registration and delivery/API panels.
- `api-openapi-schema.png` - raw OpenAPI schema endpoint from the running FastAPI service.
- `api-openapi-docs.png` - same API schema capture retained as a compatibility filename for earlier documentation drafts.

The images intentionally omit browser chrome, local filesystem paths, real credentials, and one-time API/webhook secrets.
